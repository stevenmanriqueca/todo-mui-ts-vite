# Todo App
App created with yarn and vite.

- React Hooks (useState, useContext, useReducer)
- Material UI v5
- Typescript

##  Installation

Use the package manager [yarn](https://yarnpkg.com/) to install dependencies.

```bash
yarn
```

and execute

```bash
yarn dev
```
